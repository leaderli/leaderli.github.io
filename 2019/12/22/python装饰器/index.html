类装饰器

python还提供了类装饰器，会自动实例化类，并在调用方法时调用类的__call__方法。 需要注意的是，在__init__方法，使用参数时，若使用位置参数，则装饰器注解不可带参数，且位置参数有且仅有一个，此参数在运行时即为被装饰的函数。若装饰器注解使用参数，则__init__方法，需使用默认参数

根据输出结果可以看到，类只加载一次，即__init__方法只会执行一次
